توثيق واجهة برمجة التطبيقات (API)

نظرة عامة

تتيح واجهة برمجة التطبيقات إرسال رسائل واتساب برمجياً من خلال الأجهزة المتصلة. يتم استخدام مفاتيح API للمصادقة، حيث يرتبط كل مفتاح بجهاز محدد.

المصادقة

يجب إرسال مفتاح API في ترويسة الطلب باستخدام x-api-key. مثال: 

headers: {
    'x-api-key': 'YOUR_API_KEY'
}

نقاط النهاية (Endpoints)

POST /api/keys

إنشاء مفتاح API جديد

المعلمات المطلوبة:

المعلمة النوع الوصف
deviceId string معرف الجهاز المراد ربط المفتاح به
name string اسم وصفي للمفتاح

مثال الاستجابة:

{
    "apiKey": "generated_api_key",
    "deviceId": "device_id",
    "name": "اسم المفتاح"
}
GET /api/keys

جلب قائمة مفاتيح API الخاصة بالمستخدم

مثال الاستجابة:

[
    {
        "key": "api_key",
        "deviceId": "device_id",
        "name": "اسم المفتاح",
        "isActive": true,
        "createdAt": "timestamp",
        "lastUsed": "timestamp"
    }
]
DELETE /api/keys/:key

حذف مفتاح API

مثال الاستجابة:

{
    "success": true
}
POST /api/send

إرسال رسالة واتساب

المعلمات المطلوبة:

المعلمة النوع الوصف
phone string رقم الهاتف مع رمز الدولة (مثال: 972599123456)
message string نص الرسالة
name string اسم المستلم (اختياري)

مثال الاستجابة:

{
    "success": true,
    "messageId": "message_id"
}

أكواد الأخطاء

400 - طلب غير صالح
  • بيانات مفقودة أو غير صحيحة
  • رقم الهاتف أو الرسالة غير موجودة
401 - خطأ في المصادقة
  • مفتاح API غير موجود
  • مفتاح API غير صالح
  • مفتاح API غير نشط
403 - غير مصرح
  • المستخدم لا يملك صلاحية للوصول للجهاز
404 - غير موجود
  • الجهاز غير موجود
  • مفتاح API غير موجود
429 - تجاوز حد الطلبات
  • تم تجاوز الحد الأقصى للطلبات
500 - خطأ في الخادم
  • خطأ في الاتصال بالجهاز
  • خطأ في إرسال الرسالة
  • خطأ في قاعدة البيانات

أمثلة على الاستخدام

إرسال رسالة باستخدام cURL:

curl -X POST 'https://your-domain.com/api/send' \
-H 'x-api-key: YOUR_API_KEY' \
-H 'Content-Type: application/json' \
-d '{
    "phone": "972599123456",
    "message": "مرحباً!",
    "name": "محمد"
}'

إرسال رسالة باستخدام JavaScript:

fetch('https://your-domain.com/api/send', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_API_KEY'
    },
    body: JSON.stringify({
        phone: '972599123456',
        message: 'مرحباً!',
        name: 'محمد'
    })
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
